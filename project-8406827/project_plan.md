# Design Studio Portfolio Website

## 1. Project Description
設計工作室作品集網站，風格參考 ds-k.site，採用黑白極簡設計風格，大字排版，乾淨俐落的視覺呈現。目標用戶為潛在客戶與設計愛好者。

## 2. Page Structure
- `/` - Home（首頁：Hero、About、Stats、Services）
- `/project` - Project（作品集：作品列表展示）

## 3. Core Features
- [x] 黑白極簡風格設計
- [x] 大字排版 Hero 區域
- [x] 作品集展示頁面
- [ ] 頁面切換動畫
- [ ] 聯絡表單

## 4. Data Model Design
無需資料庫，使用 mock 資料展示作品集。

## 5. Backend / Third-party Integration Plan
- Supabase: 不需要
- Shopify: 不需要
- Stripe: 不需要

## 6. Development Phase Plan

### Phase 1: 首頁 + 作品集頁面
- Goal: 建立完整的兩頁網站，黑白極簡風格
- Deliverable: Home 頁面 + Project 頁面，含導航

### Phase 2: 細節優化
- Goal: 動畫效果、互動細節優化
- Deliverable: 頁面切換動畫、hover 效果
